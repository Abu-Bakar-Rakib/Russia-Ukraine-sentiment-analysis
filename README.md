# Russia-Ukraine Sentiment Analysis

A comprehensive machine learning and deep learning project for sentiment analysis on text data related to the Russia-Ukraine conflict.

## 📋 Overview

This project employs advanced Natural Language Processing (NLP) techniques to analyze and classify public sentiment regarding the Russia-Ukraine conflict. The study explores both classical Machine Learning and state-of-the-art Transformer-based Deep Learning models to understand diverse perspectives and opinions.

### Sentiment Classes
- **Pro Russian** - Sentiments supporting the Russian perspective
- **Pro Ukrainian** - Sentiments supporting the Ukrainian perspective
- **Neutral** - Objective or impartial statements

## 📊 Dataset

- **Size:** 10,000 text samples
- **Format:** CSV file
- **Features:**
  - `text`: The textual content
  - `label`: Sentiment classification (Pro Russian, Pro Ukrainian, Neutral)

## 🛠️ Technologies & Models

This project utilizes:
- **Machine Learning Models:** Traditional ML algorithms for baseline sentiment classification
- **Transformer Models:** State-of-the-art transformer-based architectures (BERT, etc.) for advanced NLP tasks
- **Python Libraries:** scikit-learn, pandas, numpy, PyTorch, Hugging Face Transformers

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- pip package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Abu-Bakar-Rakib/Russia-Ukraine-sentiment-analysis.git
cd Russia-Ukraine-sentiment-analysis
```

2. Install required dependencies:
```bash
pip install -r requirements.txt
```
